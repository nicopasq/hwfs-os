# HWFS Operating System

HuronWest Facility Services — Financial Operating System v7

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → Import Project → select this repo
3. Deploy (zero config needed)
4. Custom domain: add hwfs.huronwest.com in Vercel dashboard

## Install as Desktop App

1. Open the deployed URL in Chrome
2. Click the install icon in the address bar (or ⋮ → Install App)
3. App appears on desktop/taskbar with its own window
